React Single Page Application (SPA)



A simple React Single Page Application built using Vite and React Router, containing Home, About, and Contact pages.

This project demonstrates the core concepts of SPA navigation without page reloads.



--> Features

* Built with React
* Fast development using Vite
* Client-side routing with React Router DOM
* Pages: Home, About, Contact
* No page reloads (true SPA behavior)
* Clean folder structure



What is a Single Page Application (SPA)?



A Single Page Application loads only one HTML page initially and dynamically updates the content using JavaScript without reloading the page.



In this project:

* index.html loads once
* React controls the UI
* Navigation switches components, not pages
